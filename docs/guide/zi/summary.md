# 拆分小结
下文中的“前两笔”，遇到14种特殊字根时，用字根代替之。

![键位图](/kbd.svg)

## 独体字
- 只有一笔的：音码＋首笔＋补码＋补码
- 只有两笔的：音码＋前两笔＋补码＋补码
- 多于两笔的：音码＋前两笔＋末笔＋补码

## 合体字
把合体字两分，按后半部分是不是独体字区分：

- 是独体字的：音码＋前两笔＋后半部分的前两笔＋**末笔**
- 是合体字的：音码＋前两笔＋后半部分的前两笔＋后半部分再拆后的**第二部分的前两笔**
